When you want to share something with the community—a question, an announcement, something for sale, or anything else—you’ll create a new message. Groups.io (the software used to create Grandview Exchange) calls this a “New Topic,” but it simply means a new message. Everywhere else in the system, they use the word message, so don’t let the wording confuse you.

# Steps to Post a New Message

- **Make sure you’re already in the group or subgroup where you want to post.** 
	 - You’ll see the group name at the top of the page (for example: Grandview Exchange, Technology, TV Club, Computer/IT Club, etc.).
- **Click the “New Topic” button** 
	 - It's on the left side of the page.
	 - Remember: “New Topic” = new message.
- **Write your message**
	 - Add a Subject (like a headline).
	 - Type your message in the large box below the Subject.
- **Add one or more hashtags (categories)**
	 - Click the # Add Tags button to the right of the Subject line.
	 - Choose the hashtag(s) that match what your message is about.
	 - Hashtags help place your message into the correct category, making it easier for neighbors to find later.
- **Click “Post Message”**  
	 - Your message will appear on the bulletin board for others to read and reply to.
