- Android allows you to do this directly within the Chrome app.
- Open Chrome and go to the website.
- Tap the three vertical dots (Menu) in the top-right corner.
- Select Add to Home screen.
- Enter a name for the shortcut.
- Tap Add, then tap Add to home screen again (or drag the icon to where you want it on your grid).
