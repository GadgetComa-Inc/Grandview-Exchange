Update your display name

Your display name appears in posts, profiles, and community interactions.

Change your name

Open your profile settings and update the display name field to the preferred name. Save the change so others see the updated name in conversations.

Review before saving

Make sure the name is spelled correctly and follows the community’s naming expectations before you save it.
