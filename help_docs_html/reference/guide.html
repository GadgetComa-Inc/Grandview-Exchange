Purpose and audience

This guide is intended for anyone using the markdown-based help system. It serves as an entry point, writing guide, and support resource.

Help structure overview

The help content is organized in a clear sequence: landing page, onboarding, navigation, messaging, account, support, and reference materials. Each page should focus on a single task or concept and link related material.

Getting started

New users should begin with the main help hub, then move through the community layout and messaging sections. The guide emphasizes instructions, quick links, and easy-to-scan content.

How to organize new help content

  • Keep each page focused on one task or concept.
  • Use headings and short steps.
  • Link related pages with relative links.
  • Place new articles in the section that matches their purpose.
