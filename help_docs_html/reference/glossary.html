Glossary

Definitions of the key terms used in the help system.

FAQ

A list of frequently asked questions with concise answers.

Guide

A longer help page that explains workflows, structure, or best practices.

Index

A navigation page that lists related help content.

Markdown

A plain-text formatting language used to write help documents before being converted into HTML.

Reference page

A short page used for definitions, terms, labels, or quick lookup.

Troubleshooting

A section dedicated to common issues, likely causes, and fixes.

TOC

Table of contents; a list of sections used for quick navigation.
