Groups.io basics

Grandview Exchange runs on Groups.io, so learning a few basic terms helps the whole site feel easier to use.

Core terms

  • Groups: the main community spaces.
  • Topics: individual messages or conversations.
  • Digests: daily or summarized email versions of conversations.
  • Hashtags: tags used to organize and find related messages.
  • Subgroups: smaller communities within the main group.
  • Message threads: replies organized under a main message.

How it works

Groups.io keeps conversations organized by group, subgroup, and category. Topics and replies stack into threads, while hashtags make it easier to browse by interest.

Why it helps

Knowing these terms makes it easier to find the right place to post, understand how replies are organized, and use email or website views more effectively.
