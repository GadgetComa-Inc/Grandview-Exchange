How to post a new message

When you want to share something with neighbors, ask a question, make an announcement, or sell an item, you create a new message on Grandview Exchange.

Steps to post a new message

  1. Make sure you are in the correct group or subgroup. The group name appears at the top of the page (for example: Grandview Exchange, Technology, TV Club, Computer/IT Club).
  2. Click the "New Topic" button. This starts a new message thread. On this site, "New Topic" means "new message."
  3. Type a clear Subject. The Subject is like a headline that tells readers what your message is about.
  4. Write your message in the large box below the Subject. Include all important details so others understand your question or announcement.
  5. Click the # Add Tags button to choose hashtags. Pick the hashtag(s) that match your message topic.
  6. Click "Post Message." Your message will then be shared in the group for other members to read and reply to.

Subject and message content

A good Subject helps people decide whether to read your message. Keep it short and specific. For example:

  • "Dining room chair for sale"
  • "Question about laundry room hours"
  • "Movie night this Friday"

In the message box, describe what you want to share, ask, or offer. If you are selling or giving away an item, include condition, price, and how someone should contact you.

Using hashtags

Hashtags place your message into the right category so other members can find it more easily. They are especially helpful when groups have many messages.

To add hashtags:

  1. Click the # Add Tags button to the right of the Subject line.
  2. Select the tags that fit your message topic.
  3. You can add more than one if the message belongs in multiple categories.
