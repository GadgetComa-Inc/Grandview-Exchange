How to post a new message

When you want to share something with the community—a question, an announcement, something for sale, or anything else—you'll create a new message. Groups.io calls this a "New Topic," but it simply means a new message.

Steps to post a new message

  1. Make sure you're already in the group or subgroup where you want to post. You'll see the group name at the top of the page (for example: Grandview Exchange, Technology, TV Club, Computer/IT Club, etc.).
  2. Click the "New Topic" button. It's on the left side of the page. Remember: "New Topic" = new message.
  3. Write your message.
    • Add a Subject (like a headline).
    • Type your message in the large box below the Subject.
  4. Add one or more hashtags (categories).
    • Click the # Add Tags button to the right of the Subject line.
    • Choose the hashtag(s) that match what your message is about.
    • Hashtags help place your message into the correct category, making it easier for neighbors to find later.
  5. Click "Post Message." Your message will appear on the bulletin board for others to read and reply to.

Understanding hashtags

Hashtags place the message into the right category so neighbors can find it later. They are especially helpful in subgroups where conversations are organized by topic. Using the correct hashtag helps people discover your message when they search or browse.
