How to post a new message

Posting in Grandview Exchange is simple. New topics are called “New Topic” in the platform, but they function like new messages in the community.

Steps to post

  1. Make sure you are in the correct group or subgroup.
  2. Click the “New Topic” button on the left side of the page.
  3. Add a subject and write your message in the editor.
  4. Add hashtags that match the topic so it is easier to find.
  5. Click “Post Message” to publish it.

Use hashtags

Hashtags place the message into the right category so neighbors can find it later. They are especially helpful in subgroups where conversations are organized by topic.
