How to attach a photo or file to a message

Residents can share photos and files directly in messages by using the attachment tools in the message editor.

Attachment steps

  1. Open a new message or reply.
  2. Use the toolbar above the message box.
  3. Click the photo button to attach an image or the paperclip button to attach a file.
  4. Drag the file into the popup window or click to choose it from your device.
  5. Click “Add” to attach it to the message.

Adding the file

The attachment window lets you drag and drop or browse for the file. Once added, it appears with the message so other residents can view or download it.
