Edit or delete a message

Use editing and deletion tools to correct a message or remove something that should not remain published.

Edit a message

If you posted something that needs a quick correction, open the message and use the edit option to update the content before saving your changes.

Delete a message

If a message should be removed, use the delete option available on your post. Review the message before deleting it so you do not remove something that others still need to read.
