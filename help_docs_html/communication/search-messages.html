How to search for messages

Use search and tags to quickly find old posts, specific topics, and conversations in the right group.

Search basics

Start with the search box in the group or site. Search by words, names, topics, or keywords that match your question.

Filters and tags

Use hashtags and category filters to narrow a search to a specific theme, subgroup, or interest area.

Search tips

  • Search for one main keyword first.
  • Use a tag if you already know the topic category.
  • Browse older threads when the exact post title is unknown.
