How to attach a photo or file to a message

When creating or replying to a message, you can share a photo or file by attaching it to your post.

Steps to attach a photo or file

  1. While writing a new message or replying to one, use the toolbar just above where you write your message.
    • To attach a photo, click the photo button.
    • To attach a file, click the paper clip button.
  2. Drag the photo or file onto the popup window that appears, or click the link to choose photos or files.
  3. Click Add to attach the photo or file. The attachment will appear with your message when posted.

About attachments

The attachment window lets you drag and drop or browse for the file. Once added, it appears with the message so other residents can view or download it. Keep file sizes reasonable so the message sends quickly and is easy for others to download.
